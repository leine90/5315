What is Docker?

Docker is a development tool that facilitates work flow by packing applications using containers. Containers are smaller than virtual machines and are open source. One of the major benefits of containers is that they contain their own tools to operate indepently of extensive software libraries which allows for easier deployment

What is Nginx?

NGINX is another open source resource for web serving known for its speed,stabillity, and the abillity to handle the capacity of a growing internet. NGINX was originally developed in 1999 and opensourced in 2004 which allowed to grow into its current form today.

- https://www.nginx.com/resources/glossary/nginx/

What is VM (Virtual Machine)?

A virtual machine is a file that acts like an actual computer. Running in a window like a program can give the same experience as the host computer. A major benefit of a VM enviroment is that it is seperate from the rest of the host computer which creates a good space to test applications. Growing up virtual machines were used to run retro video games on the computer.

- https://azure.microsoft.com/en-us/overview/what-is-a-virtual-machine/

Who is Cloud Computing Service Provider - Digitalocean

Digital Ocean is a cloud platform meant for developing software with 12 data centers and two campus locations on the east coast. Digital Ocean is an alternative to hosting your website via AWS or Microsoft's Azure providing VMs that they call droplets
